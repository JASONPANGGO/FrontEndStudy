# 匹配以特定字母开头或结尾的单词
```javascript
// 匹配以s开头以e结束的单词
// \b 代表单词的开始或结束
// \S 代表任意非空字符串
// 而至于* 自然是表示匹配任意多个字符。
// 而使用\S*? 是为什么呢，为什么要加？ 呢？这就涉及到正则表达式的懒惰模式了。
// 如果使用.* 表示的就是贪婪模式，而.*? 表示的就是懒惰模式。
// 贪婪模式下会尽量匹配最长的字符串，而懒惰模式会尽量匹配最短的字符串。
// 举个栗子：
// 对于字符串abcgabc

// 贪婪模式– a.*c –得到的答案为：abcgabc 
懒惰模式– a.*?c –得到的答案为：abc,abc
let pattern = /\bs\S*?e\b/g

let string = 'site sea sue sweet see case sse ssee loses'

```
